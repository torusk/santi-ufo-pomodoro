# 🛸 三体UFO吸引機 (Santi UFO Pomodoro)

このプロジェクトは、ポモドーロタイマーの終了時にUFOが飛来し、ブラウザ上のコンテンツを「归零（帰零）」処理（吸い上げ）てしまうChrome拡張機能です。

## 🚀 特徴
- **タイマー機能**: ポップアップから時間を設定してカウントダウンを開始。
- **偵察飛行**: タイマー稼働中、UFOが画面上を不規則に偵察飛行します。
- **帰零処理**: タイマー終了時、UFOが画面中央上部に静止し、ページ内の要素を画面下部から順番に一つずつ吸い上げます。
- **世界観**: 三体（Three-Body Problem）の高度文明をイメージした無機質かつ強力な演出。

## 🛠 インストール方法
1. このリポジトリをクローンまたはダウンロードします。
2. Google Chromeで `chrome://extensions` を開きます。
3. 「デベロッパーモード」をオンにします。
4. 「パッケージ化されていない拡張機能を読み込む」をクリックし、このプロジェクトのフォルダを選択します。

## 📖 使い方
1. 拡張機能のアイコンをクリックしてポップアップを開きます。
2. 時間を選択して「開始」を押します。
3. タイマーが終了すると、帰零処理が開始されます。
4. 処理完了後、「重构」ボタンでページを再読み込みできます。

## ⚠️ 注意事項
- この拡張機能はエンターテインメント目的です。作業中の未保存のデータがあるページでの使用にはご注意ください。
